## 🧪 Projects

### 🔹 MCP + IA Integration
Integración de IA mediante Model Context Protocol (MCP) en un entorno local seguro.
- **Problema:** Necesidad de interactuar con servicios internos y mejorar la eficiencia del desarrollo sin exponer datos sensibles.
- **Solución:** Desarrollo de un servidor MCP local para conectar asistentes de IA con APIs internas y bases de datos.
- **Impacto:** Reducción drástica de tiempos de desarrollo, mejora de capacidades operativas y establecimiento de nuevas integraciones de IA.
